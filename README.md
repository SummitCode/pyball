# pyball

Library for grabbing baseball statistics in python

## Requirements
- Python 3(.7)

## Install/Build From Source
```
pip install --user --upgrade setuptools

cd pyball

setup.py install
```

Read the [wiki](https://github.com/SummitCode/pyball/wiki) for tutorials on using pyball.
